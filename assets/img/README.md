# assets/img/

Place your images here. Recommended files:

| File | Description | Recommended size |
|---|---|---|
| `avatar.jpg` | Your profile photo (shown in author sidebar) | 400 × 400 px |
| `header-bg.jpg` | Optional dark banner for the home page header | 1920 × 600 px |

Tips:
- Keep images under 500 KB for fast page loads — use tinypng.com to compress
- Use `.jpg` for photos, `.png` for screenshots and diagrams
- Sanitise any screenshots before uploading — remove IPs, hostnames, internal details
